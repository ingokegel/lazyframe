Lazyframe

Dependency-free library for lazyloading iframes

https://vb.github.io/lazyframe/

How it works

Lazyframe creates a responsive placeholder for embedded content and only requests it when the user interacts with it. This decreases inital page load time. Lazyframe comes with brand-like themes for YouTube, Vimeo and Vine.

Examples
